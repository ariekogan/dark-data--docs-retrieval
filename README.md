- **docs-retrieval** — Users have large collections of documents in Dropbox (handbooks, reports, notes, papers) and need natural-language retrieval with citations. Browsing is too slow; keyword search on filenames misses semantic matches. The assistant should ingest folders on request, search semantically across indexed content, and ground its answers in retrieved chunks.
